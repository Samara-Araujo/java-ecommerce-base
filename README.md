🛒 E-commerce Simulation - Java & Web
Projeto desenvolvido para aplicar conceitos de Back-end e Front-end de forma integrada.

📁 Estrutura do Projeto:
/backend: Contém a lógica de negócios em Java, incluindo a classe Produto e cálculos de desconto.

/frontend: Interface moderna em HTML5 e CSS3, focada na experiência do usuário de e-commerce.

🚀 Funções adicionadas :

[] campo de cupom de desconto.

[] Listar múltiplos produtos dinamicamente.

[] Integrar JavaScript para mostrar o cálculo do Java na tela.
